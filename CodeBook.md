# Codebook - Daily activities sensor data set

# grouped_mean_dataset: Merged test and training data sets containing mean() and std() of the variables

1. subject_id : Unique identifier of the subject performing the activity.
    * Type: Integer
    * Valid range: 1-30

2. activity : Identifies the activity that subject performed when the data observation was recorded.
    * Type: String
    * Valid range: (WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING)
